 CCodeInspectionCodeInspection is a Handler for one File or Text to search, read and write. It can be handeld like a LinkedList. When you Search for a Match the code will be splittet and added to a LinkedList you then receive a Node with which you can change the Code. You create CodePieces from the CodeInspection to then change the underlining File content or Text with the adjusted CodePiece. If you are in READ_WRITE mode there can always be only one CodePiece active.
